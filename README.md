# GROK MINI

**GROK MINI** is an advanced geopolitical risk analysis and prediction platform powered by AI-driven neural simulations. It provides real-time intelligence summaries, impact assessments, and cascading predictions for complex global scenarios.

## Features

- 🧠 **AI-Powered Intelligence Analysis** - Neural simulation models for geopolitical scenario analysis
- 📊 **Advanced Visualizations** - Interactive charts, radar plots, and timeline projections
- 🎯 **Cascading Predictions** - Multi-stage impact and risk predictions
- 🔐 **Secure Authentication** - User authentication and role-based access control
- 📱 **Responsive Design** - Mobile-optimized interface with real-time updates
- ⚡ **High Performance** - Built with React, Vite, and modern web technologies

## Tech Stack

- **Frontend**: React 18, Vite, TypeScript
- **UI Components**: Recharts, Framer Motion, Lucide Icons
- **State Management**: React Query (TanStack Query)
- **Routing**: React Router
- **Styling**: Tailwind CSS
- **Build Tool**: Vite

## Getting Started

### Prerequisites
- Node.js 16+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/kharadearjun98-web/GROK-.git
cd GROK-

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
npm run preview
```

## Environment Variables

Create a `.env` file in the root directory:

```

VITE_AUTH_TOKEN=your_auth_token
```
